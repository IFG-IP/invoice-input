# 請求書入力ツール 簡易操作マニュアル

## 1. 書類の取り込み

1. 画面のアップロードエリアに請求書・領収書のPDFまたは画像を追加します。
2. フォルダ単位、または複数ファイルをまとめて追加できます。
3. 追加後、ファイル一覧に読み込み状況が表示されます。
4. 不要なファイルは一覧の `削除` で外せます。

## 2. AI抽出

1. 書類を追加したら `次へ` を押します。
2. AIがExcelフォーマットの項目に合わせて内容を抽出します。
3. 処理が終わると `目視確認` 画面に進みます。

## 3. 目視確認と修正

1. 左側に請求書・領収書のプレビューが表示されます。
2. 右側に抽出データの編集フォームが表示されます。
3. 画面上部のファイル一覧から確認する書類を切り替えます。
4. 現在編集中のファイルは、ファイル一覧で太い枠で囲まれます。
5. 右側の `抽出データ` 欄にも、現在編集中のファイル名が表示されます。
6. 抽出内容を手で修正すると、自動で反映されます。
7. `支払日` はクリックするとカレンダーから選択できます。

## 4. 必須項目の確認

未入力の必須項目は赤枠で表示されます。

主な必須項目は以下です。

- 受領方法
- 支払先名
- 支払金額(税抜)
- 支払内容
- 支払方法
- 支払日
- 利用月
- 合計支払金額
- 銀行名
- 支店名
- 口座種別
- 口座番号
- 口座名義フリガナ
- 請求書発行日

請求書に記載がなく入力できない項目がある場合は、skyberry出力時に確認画面が表示されます。内容を確認し、問題なければ `skyberry出力する` を押します。

## 5. skyberry Excel出力

1. すべての書類を確認します。
2. `skyberry Excel出力` を押します。
3. 未入力の必須項目がある場合は確認画面が表示されます。
4. 問題なければExcelファイルがダウンロードされます。
5. 出力ファイル名は `skyberry-import-YYYYMMDD-HHMM.xlsx` です。

## 6. 注意事項

- 元のExcelファイルは直接上書きされません。
- 出力されるExcelは新しいファイルとしてダウンロードされます。
- 手修正した内容は、`skyberry Excel出力` を押した時点で反映されます。
- ブラウザを更新すると画面上の作業状態が消えるます。出力前に必要な確認を済ませてください。
