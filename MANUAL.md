# 請求書入力ツール 操作マニュアル（依頼者用）

このマニュアルは、請求書・領収書を取り込み、AI抽出結果を確認して、skyberry取込用Excelを出力する依頼者向けの手順書です。

## 1. 操作方法

### 画面を開く

管理者から共有されたURLがある場合は、そのURLをブラウザで開きます。

ローカル環境で起動する場合は、このプロジェクトフォルダでPowerShellを開き、以下を実行します。

```powershell
powershell -NoProfile -ExecutionPolicy Bypass -File .\serve.ps1
```

起動後、ブラウザで以下を開きます。

```text
http://127.0.0.1:5173/
```

### 書類を追加する

画面左側の「請求書・領収書アップロード」から書類を追加します。

| 操作 | 内容 |
| --- | --- |
| `ファイルを選択` | PDFや画像を複数選択して追加します。 |
| `フォルダを選択` | フォルダ内の対応ファイルをまとめて追加します。 |
| ドラッグ&ドロップ | ファイルまたはフォルダをアップロードエリアへ直接置きます。 |
| `削除` | 選択済み一覧から対象ファイルだけ外します。 |
| `すべて削除` | 選択済みファイルをすべて外します。 |

対応ファイルは、PDF、JPG、JPEG、PNG、HEIC、HEIFです。

### AI抽出へ進む

書類を追加したら、`次へ` を押します。AIがExcelフォーマットの項目に合わせて内容を抽出します。

処理中は画面に進捗が表示されます。複数ファイルを投入した場合は、1件ずつ順番に処理されます。

### 目視確認で内容を修正する

AI抽出が終わると、`目視確認` 画面に進みます。

| 画面エリア | 内容 |
| --- | --- |
| 上部のファイル一覧 | 確認する書類を切り替えます。未入力の必須項目がある書類は赤系の表示になります。 |
| 左側プレビュー | 請求書・領収書の画像またはPDFプレビューを確認します。 |
| 右側の抽出データ | AIが抽出した値を確認し、必要に応じて手修正します。 |
| `戻る` | アップロード画面へ戻ります。 |
| `skyberry Excel出力` | 確認済みデータをExcelとしてダウンロードします。 |

入力欄を修正すると、内容は画面上のデータに自動反映されます。`支払日` はカレンダーから選択できます。

### Excelを出力する

すべての書類を確認したら、`skyberry Excel出力` を押します。

未入力の必須項目がある場合は確認画面が表示されます。入力画面に戻って修正する場合は `入力画面に戻る`、未入力のまま出力してよい場合は `skyberry出力する` を押します。

AI抽出できていない書類がある場合も確認画面が表示されます。未抽出の書類はExcelに出力されないため、内容を確認してから `このまま出力する` を押してください。

出力ファイル名は以下の形式です。

```text
skyberry-import-YYYYMMDD-HHMM.xlsx
```

## 2. 利用手順

1. 請求書・領収書のPDFまたは画像を準備します。
2. ツール画面をブラウザで開きます。
3. `ファイルを選択`、`フォルダを選択`、またはドラッグ&ドロップで書類を追加します。
4. 選択したファイル一覧に、投入した書類が表示されていることを確認します。
5. 不要なファイルがある場合は `削除` または `すべて削除` で外します。
6. `次へ` を押してAI抽出を開始します。
7. `目視確認` 画面で、書類ごとにプレビューと抽出データを照合します。
8. 金額、支払先、支払日、口座情報など、重要項目を必ず確認します。
9. 誤りや空欄があれば、右側の入力欄で修正します。
10. すべての書類を確認したら、`skyberry Excel出力` を押します。
11. 必須項目や未抽出書類の確認画面が出た場合は、内容を確認して出力可否を判断します。
12. ダウンロードされたExcelをskyberry取込用ファイルとして利用します。

## 3. 必ず確認する項目

AI抽出結果は必ず目視確認してください。特に以下の項目は、申請・支払処理への影響が大きいため重点的に確認します。

- 受領方法
- 支払先名
- 支払金額（税抜）
- 合計支払金額
- 支払内容
- 支払方法
- 支払日
- 利用月
- 請求書発行日
- 請求書番号
- 銀行名
- 支店名
- 口座種別
- 口座番号
- 口座名義フリガナ

未入力の必須項目は赤枠で表示されます。書類上に記載がない項目は、社内ルールに従って入力するか、未入力のまま出力してよいか確認してください。

## 4. 想定エラー・注意事項

| 状況 | 主な原因 | 対応 |
| --- | --- | --- |
| 画面が開かない | ローカルサーバーが起動していない、URLが違う | PowerShellで `serve.ps1` が起動しているか確認し、`http://127.0.0.1:5173/` を開き直してください。 |
| ファイルを追加できない | 対応外形式のファイルを選んでいる | PDF、JPG、JPEG、PNG、HEIC、HEIFのいずれかに変換してから追加してください。 |
| `次へ` を押せない、または進めない | 書類が未選択 | アップロードエリアに書類を追加してください。 |
| AI抽出でエラーになる | API接続エラー、設定不備、通信不安定 | しばらく待って再実行してください。解消しない場合は管理者に連絡してください。 |
| `APIに接続できません` と表示される | ローカル起動漏れ、または共有環境のAPI設定不備 | ローカルの場合は `serve.ps1` を起動してください。共有URLの場合は管理者に連絡してください。 |
| PDFのプレビューが表示されない | PDF読込失敗、PDF.js読込失敗、特殊なPDF形式 | ブラウザを再読み込みし、再投入してください。改善しない場合はPDFを画像化して投入してください。 |
| HEIC/HEIFが読めない | HEIC変換ライブラリの読込失敗、ブラウザ相性 | JPGまたはPNGに変換してから再投入してください。 |
| 抽出結果が空欄になる | 書類の文字が不鮮明、項目が書類にない、AIが判定できない | プレビューを見ながら手入力してください。 |
| 金額や日付が違う | 書類内に複数の候補がある | 原本のラベルを確認し、正しい欄に修正してください。 |
| 未入力の必須項目があります | 必須項目が空欄 | 赤枠の項目を入力してください。記載がない場合は、出力してよいか社内ルールに従って判断してください。 |
| 出力されない書類があります | AI抽出失敗、読込失敗、プレビューなし | 入力画面に戻って確認します。問題なければ、抽出済みの書類だけで出力できます。 |
| Excelがダウンロードされない | ブラウザのダウンロード制限、テンプレート読込失敗 | ブラウザのダウンロード許可を確認し、再度 `skyberry Excel出力` を押してください。 |

### 注意事項

- AI抽出結果は参考値です。必ず原本と照合してからExcel出力してください。
- ブラウザを更新すると、アップロード済みファイルや修正中の内容は消えます。
- 元のExcelテンプレートは直接上書きされません。出力時は新しいExcelファイルがダウンロードされます。
- 手修正した内容は、`skyberry Excel出力` を押した時点の内容でExcelに反映されます。
- PDFの画像プレビューは先頭12ページまでが対象です。PDF内テキストは最大25ページ分をAI確認に利用します。
- インターネット接続が不安定な場合、PDF表示、HEIC変換、Excel出力ライブラリの読込に失敗することがあります。
- 同じ書類を複数回投入すると、重複してExcelに出力される可能性があります。選択済み一覧で不要な重複を削除してください。
- 機密情報を含む書類を扱うため、出力Excelやダウンロードファイルの保存先に注意してください。
